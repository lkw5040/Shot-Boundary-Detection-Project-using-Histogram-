# Shot-Boundary-Detection-Project-using-Histogram-
경희대학교 영상분석과 패턴인식 수업에서 진행한 히스토그램을 이용한 Shot Boundary Detection Project에 대한 코드
본 프로젝트의 코드는 경희대학교 소프트웨어융합대학 이대호 교수님의 KhuCV 툴을 사용해서 만든 코드이며 실습으로 진행한 프로젝트
프로젝트와 관련해서 전체적인 코드는 KhuCV를 통해서 접근하고, 본 계정에서는 수정한 코드 일부만 업로드 하였음.

Abstract
본 분석에서는 영상이 재생될 때 장면이 변화하는 것을 영상분석과 패턴인식의 방식으로 분석해보는 것이며 조금 더 자세하게는 히스토그램 유사도를 이용하여 Shot Boundary Detection을 해보는 것이다. 본 과정은 경희대학교 소프트웨어융합대학의 영상분석과 패턴인식 강의에서 배운 내용을 토대로 진행하며 경희대학교 이대호 교수님께서 제공해주신 KhuCV 툴을 활용하여 분석했다. 히스토그램의 값을 이용해 각각 Cross Correlation, Intersection, KLD의 방식으로 유사도를 측정하고 이것을 특정 임계값을 이용해서 각각 Precision, Recall, f1 score의 방식으로 정량적으로 영상 분석 결과를 해석한다. 또한 위의 3x3의 실험 결과를 해석하고 성능을 개선할 수 있는 더 나은 방법을 모색하고 더 나아가 영상과 히스토그램간의 관찰을 통해 얻은 다양한 특징들을 해석하고 기록한다.
